# FMS-FSDP

FMS-FSDP is a scalable model training framework developed by foundation model stack. The repo here modifies it s.t it fits the needs for training pruned models.

## Install
```bash
bash fms-fsdp/scripts/install.sh
```

## Training from scratch
```bash
bash fms-fsdp/scripts/training.sh
```

## Fine-tuning
```bash
bash fms-fsdp/scripts/hf2fms.sh
bash fms-fsdp/scripts/finetuning.sh
```
